# Dia Browser — Layout Patterns

## Page Structure
- Fixed centered nav in rounded pill container (rounded-16, border-black/8, shadow)
- Full-bleed hero with centered text, circular background motif
- 12-column grid system for feature sections
- Sticky left nav + scrolling right content panel
- Bento grid: mixed col-span-6 / col-span-4 cards
- Centered CTA block (max-w-760)

## Spacing Scale
- Micro: 8px, 10px, 11px gaps
- Small: 12px, 16px, 20px gaps
- Medium: 40px, 48px, 60px vertical
- Large: 80px, 100px, 120px section margins
- Card padding: 20–30px

## Border Radius
- Small: 5px (avatars), 6px
- Medium: 12px (cards), 14px (buttons/inputs)
- Large: 16px (nav), 20px (feature cards), 24px
- Full: pills, dots, badges

## Container Patterns
- Glass panels: semi-transparent white (70%), blur, subtle border
- Cards: light fill, 1px border at 5–8% black opacity
- Sections: full-width, content max-width constrained
- Floating elements: offset shadows, slight scale transforms

## Grid Strategy
- Hero: centered single-column
- Features: 4+8 split (nav + content)
- Bento: 6+6 and 6+4 card combos
- Stats: equal-width columns
- Footer: multi-column link grid
